# FirmRunner

FirmRunner is an early-stage system designed to reduce administrative work in accounting firms.

The goal is to handle intake, documents, deadlines, billing, and reporting through a single system.

## Current Status

- In development
- Early access phase
- Collecting feedback from accounting firms

## Problem

Accounting firms spend too much time on non-accounting work.
Intake is messy.
Documents are scattered.
Deadlines are missed.
Billing is slow.

## Approach

Vertical AI.
Agents that understand accounting workflows.
Automation that feels like an employee.

## Tech Stack

- React
- Tailwind CSS
- Supabase
- Gemini API

## Setup

1. Clone repo
2. npm install
3. Setup .env (see .env.example)
4. npm run dev

## Environment Variables

- VITE_SUPABASE_URL
- VITE_SUPABASE_ANON_KEY
- GEMINI_API_KEY

## Notes

This is a pre-seed stage prototype.
Focus is on clarity of thinking, not fancy UI.
